解决pip安装Python库时：ReadTimeoutError

2020-03-19

近来Python不知怎的就被墙了，据说是有一个非法网站跟Python的网站很像，本来想禁那个非法网站的，结果~~😢😢😢

所以我们如果再用pip安装Python的某些库时，就会出现连接不上的问题。

解决办法

指定国内的下载源来进行下载，把原来的安装命令替换为：

 pip install -i https://pypi.douban.com/simple 需要安装的Python库

试了试，速度还挺快的。😁
